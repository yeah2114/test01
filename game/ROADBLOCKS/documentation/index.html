How to ?

How to play ?

Road block is puzzle game in which you need to create destination path by drag and drop on accurate location.

you drag road block and put it on correct path.

you have different levels to play in game with different difficulty, unlock new levels and gain score.

 

 

How to live game on server ?

You need a domain and hosting then you can make this game live on internet and can enjoy game.

Game is ready just copy the index.html and game folder in your root path or any subdomain or subpath of your server then visit your domain , thats it your game is live and ready to play. :)

 

 

How to Customize ?

Game is ready and you dont need to make changes but still you want make changes in game then you have three file index.html , index.js and style.css, you can follow the below customization guide and can make changes according you want.
